# 📄 AI Resume Screening & Candidate Ranking Tool

## Overview

An AI-powered Resume Screening and Candidate Ranking application that analyzes multiple resumes against a job description using Natural Language Processing (NLP). The system automatically ranks candidates based on semantic similarity to help recruiters reduce manual screening effort.

---

## Features

- Upload Job Description
- Upload Multiple Resume PDFs
- Resume Text Extraction
- Text Preprocessing
- Semantic Similarity Matching *(Coming Soon)*
- Candidate Ranking *(Coming Soon)*
- CSV Export *(Coming Soon)*
